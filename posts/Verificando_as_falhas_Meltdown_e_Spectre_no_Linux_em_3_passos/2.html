Verificando as falhas Meltdown e Spectre no Linux em 3 passos

Atenção

Este script só funciona adequadamente em máquinas físicas já que máquinas virtuais não permitem o acesso ao firmeware.

3 passos:

1) Abra o seu terminal e digite o seguinte comando para baixar o arquivo compactado com a ferramenta de verificação da Intel:

cd ~/Downloads && wget https://downloadmirror.intel.com/27150/eng/SA00086_Linux.tar.gz

2) Em seguida aperte enter e digite o seguinte comando para extrair os arquivos:

mkdir intel && tar xvzf SA00086_Linux.tar.gz -C intel && cd intel

3) Na sequência digite o seguinte comando para executar a ferramenta como super usuário e aperte enter:

sudo python intel_sa00086.py
