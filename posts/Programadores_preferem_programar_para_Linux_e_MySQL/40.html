Programadores preferem programar para Linux e MySQL


Programadores preferem Linux e MySQL segundo pesquisa do Stack Overflow
Recentemente, o portal Stack Overflow publicou um conjunto de estatísticas que mostram claramente que os programadores programam cada vez mais para Linux. Vamos conhecer o resultados.

O Stack Overflow é um portal direcionado para programadores, onde estes colocam perguntas e aguardam que a comunidade lhes responda. Segundo dados recentes, este portal tem mais de 100 mil programadores por todo o mundo.

Programadores programam cada vez mais para Linux?

Para mostrar as tendências dos programadores, o portal disponibilizou recentemente um conjunto interessante de estatísticas que obteve dos seus utilizadores. Um dos dados que mais se destaca é o facto dos programadores estarem a desenvolver cada vez mais para plataformas Linux.

Como se pode ver pelos dados seguintes, 48,3% dos respondentes já desenvolveram para Linux e “apenas” 35,4% dos respondentes desenvolveram para Windows. Podem acessar todos os dados aqui.
Ao nível dos sistemas de gestão de base de dados (SGBD), o MySQL e SQL Server continuam a ser os preferidos dos programadores segundo as estatísticas do portal.
No que diz respeito aos IDEs, os mais populares entre os programadores são o Visual Studio Code, o Visual Studio e o popular editor de texto Notepad++.
Neste link podem encontrar outras estatísticas interessantes. O Linux é sem dúvida uma plataforma imprescindível para a computação mundial. Ao nível dos ambientes domésticos e também do IoT, este sistema tem ganho cada vez mais adeptos e os resultados falam por si.
